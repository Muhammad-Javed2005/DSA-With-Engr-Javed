# Vectors (Dynamic Arrays)

Welcome to the **Vectors** module! This section explores dynamic arrays, allowing automatic resizing when elements are added or removed.

## 🎯 Overview
Vectors (in C++) or dynamic arrays dynamically manage their internal capacity. They abstract the complexities of resizing and copying, providing a seamless linear data structure for scalable applications.

## 📚 What You Will Learn
* **Fundamentals:** Dynamic sizing, capacity vs. size, and reallocation.
* **Standard Library:** Utilizing STL vector methods (`push_back`, `pop_back`, `insert`, `erase`).
* **Memory Management:** How capacity doubles and its amortized time complexity.
* **Vector vs Array:** Understanding when to choose a dynamic array over a static array.

## 🚀 Getting Started
1. Practice initializing and modifying vectors using STL.
2. Understand the concept of "Amortized O(1) Time Complexity".
3. Solve problems that require dynamic data limits.
