# Binary Trees

Welcome to the **Binary Trees** module! This section branches out into hierarchical data structures.

## 🎯 Overview
A Binary Tree operates hierarchically—each node possesses at most two children (left and right). They simulate natural hierarchies, allow rapid searches, and are fundamental for advanced hierarchical structures.

## 📚 What You Will Learn
* **Fundamentals:** Nodes, roots, leaves, height, and depth.
* **Traversals:** Inorder, Preorder, Postorder (DFS), and Level-order (BFS).
* **Properties:** Complete, Full, Perfect, and Balanced binary trees.
* **Algorithms:** Finding LCAs, tree diameters, and mirroring trees.

## 🚀 Getting Started
1. Relate recursive algorithms naturally to tree traversal logic.
2. Understand the shift from linear processing to branching pathways.
3. Code recursive DFS iteratively using Stacks.
