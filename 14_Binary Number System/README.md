# Binary Number System and Bit Manipulation

Welcome to the **Binary Number System** module! This section goes deeply into how computers fundamentally process numbers and how to leverage bitwise operations for massive optimizations.

## 🎯 Overview
Everything on a computer boils down to 1s and 0s. Understanding binary systems and Bit Manipulation enables algorithms that execute at low-level CPU speeds with minimal memory footprints.

## 📚 What You Will Learn
* **Fundamentals:** Base-2 conversions, 1's and 2's complement representations.
* **Bitwise Operators:** AND (`&`), OR (`|`), XOR (`^`), NOT (`~`), Left Shift (`<<`), Right Shift (`>>`).
* **Masking:** Checking, setting, clearing, and toggling specific bits.
* **Patterns:** Power of two checks, counting set bits (Brian Kernighan's Algorithm), XOR tricks.

## 🚀 Getting Started
1. Memorize truth tables for fundamental bitwise operators.
2. Practice representing decimal values visually as 32-bit blocks.
3. Explore the magical properties of XOR (`a ^ a = 0`).
