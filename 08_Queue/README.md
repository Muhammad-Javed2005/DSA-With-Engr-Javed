# Queues Data Structure

Welcome to the **Queues** module! This section covers the FIFO (First-In-First-Out) principle and robust scheduling algorithms.

## 🎯 Overview
A Queue is a linear data structure wherein the first element added is the first one to be removed. It is heavily utilized in operating systems, printer spooling, and Breadth-First Search (BFS) algorithms.

## 📚 What You Will Learn
* **Fundamentals:** `enqueue()`, `dequeue()`, `front()`, and `rear()` operations.
* **Implementations:** Using Arrays (Circular Queue) and Linked Lists.
* **Variants:** Double-ended Queues (Deque), Priority Queues.
* **Applications:** Tree level-order traversal, sliding window maximums.

## 🚀 Getting Started
1. Grasp the circular array mechanism for efficient static queues.
2. Implement a queue class from scratch.
3. Understand how Deque optimizes specific two-pointer/window problems.
