## 📌 Detailed Topics Covered

### 🔹 1. String Functions

Learn basic built-in string operations: - `length()` - `size()` -
`append()` - `substr()` - Character access

------------------------------------------------------------------------

### 🔹 2. Palindrome Check

A string is called a palindrome if it reads the same forward and
backward.

**Logic Used:** - Compare characters from start and end - Move towards
center

**Example:**

    Input:  madam
    Output: Palindrome

------------------------------------------------------------------------

### 🔹 3. Anagram Check

Two strings are anagrams if they contain the same characters with the
same frequency.

**Logic Used:** - Sort both strings OR - Use frequency array

**Example:**

    Input:  listen, silent
    Output: Anagram

------------------------------------------------------------------------

### 🔹 4. Reverse String

Reverses the given string.

**Logic Used:** - Two-pointer approach OR - Built-in reverse function

**Example:**

    Input:  hello
    Output: olleh

------------------------------------------------------------------------

### 🔹 5. Remove Spaces

Removes all spaces from a string.

**Logic Used:** - Traverse string - Ignore spaces

**Example:**

    Input:  "hello world"
    Output: "helloworld"

------------------------------------------------------------------------

### 🔹 6. String Compression

Compress string using character count.

**Logic Used:** - Count consecutive characters - Append char + count

**Example:**

    Input:  aaabbc
    Output: a3b2c1
