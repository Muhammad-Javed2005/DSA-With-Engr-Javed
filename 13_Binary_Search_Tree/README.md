# Binary Search Trees (BST)

Welcome to the **Binary Search Trees (BST)** module! This section combines the flexibility of linked lists with the fast lookup power of binary search.

## 🎯 Overview
A BST imposes a strict condition structure on binary trees: the left subtree contains values lesser than the root, and the right subtree contains values greater. This property guarantees rapid searching, insertion, and deletion behaviors.

## 📚 What You Will Learn
* **Fundamentals:** BST property, balanced operations.
* **Operations:** Searching, inserting, computing boundaries, and deleting nodes.
* **Inorder traversal:** Property extraction (inorder of a BST gives sorted output).
* **Validation:** Checking algorithmically if a generic tree is a valid BST.

## 🚀 Getting Started
1. Master the standard deletion cases (0, 1, or 2 children).
2. Use the "min/max boundary" method to validate BST properties.
3. Combine BST logic with advanced tree modifications.
